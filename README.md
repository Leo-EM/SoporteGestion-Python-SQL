# WorkFlow IT Management System

A professional management system for IT technical support, developed in **Python** with **PostgreSQL**. This project features a modular architecture and a graphical user interface (GUI) to manage clients, services, and work orders.

## 🚀 Features
- **Client Management:** Full CRUD (Create, Read, Update, Delete) functionality.
- **Relational Database:** Robust data integrity using PostgreSQL.
- **Modular Design:** Clear separation between UI, Business Logic, and Database connection.
- **Security:** SQL Injection prevention using parameterized queries.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Database:** PostgreSQL 17
- **GUI:** Tkinter (Migrating to CustomTkinter)
- **Driver:** Psycopg2

## 📝 How to use
1. Clone the repository.
2. Install dependencies: `pip install psycopg2-binary`.
3. Configure your database credentials in `Database/conexcion.py`.
4. Run `conexion_db.py` to start the application.
